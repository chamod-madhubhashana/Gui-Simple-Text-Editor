# Simple Text Editor  

A basic text editor built using Java Swing that supports essential file operations.  

## Features  
 ● Create new text files  
 ● Open existing text files  
 ● Save text files  

## Requirements  
 ● Java (JDK)  
 ● IDE like NetBeans
